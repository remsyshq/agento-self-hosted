Agento

Self-hosted AI agent platform

Install
curl -fsSL https://get.agento.host/install.sh | bash

Then run

1. agento init — set up admin account
2. agento start — start the platform
3. agento open — open dashboard in browser

Requires Docker (OrbStack or Docker Desktop), Node.js 20+, and macOS or Linux.
